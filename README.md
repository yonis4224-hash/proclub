# My First Flutter App (تطبيقي الأول)

هذا مشروع بداية بسيط لتطبيق Flutter — قائمة مهام صغيرة.

المتطلبات:
- تثبيت Flutter SDK: https://flutter.dev
- محرر مثل VS Code أو Android Studio
- محاكي Android أو جهاز حقيقي

الأوامر الأساسية:
1. إنشاء مشروع جديد (محليًا): `flutter create my_app`
2. انسخ محتوى `lib/main.dart` بهذا الملف.
3. ادخل للمجلد: `cd my_app`
4. شغّل التطبيق: `flutter run`

ملف البداية: `lib/main.dart` يحتوي على واجهة بسيطة لإضافة وحذف مهام من الذاكرة (بدون تخزين دائم).
